## Resumen teórico
La programación orientada a objetos es un paradigma de programación que organiza el desarrollo de software alrededor de objetos. Un objeto representa una entidad que combina datos y comportamiento.

Entre los conceptos iniciales más importantes se encuentran:

### Clase
Una clase es una plantilla o modelo a partir del cual se crean objetos. Define atributos y métodos comunes.

### Objeto
Un objeto es una instancia concreta de una clase. Tiene un estado y puede ejecutar acciones definidas en sus métodos.

### Atributos
Los atributos representan las características o propiedades de un objeto.

### Métodos
Los métodos representan las acciones o comportamientos que un objeto puede realizar.

### Importancia de este enfoque
La programación orientada a objetos permite organizar mejor el código, reutilizar componentes y modelar problemas de forma más cercana a la realidad.
